Detailed Description

Shapes' dimensions modifiable at run-time, unlike a typical Mesh whose dimensions are stored in a file and loaded into memory. Shape is simply a base class from which instantiable sub-classes such as Cube or Sphere inherit. Each shape type implements unique interfaces for modifying or accessing its data.

Note
Future versions may see the removal of the Shape class, as it currently implements functionality which Mesh may implement.

Constructor & Destructor Documentation

bakge::Shape::Shape ( )
protected

Shape is not explicitly instantiable. Different types of shapes such as cubes and spheres have different interfaces for setting, manipulating and accessing dimension data. As such, only these sub-classes will be able to create new instances through appropriate factory functions.
