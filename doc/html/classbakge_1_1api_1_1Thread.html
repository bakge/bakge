Detailed Description

Threads (also called lightweight processes) are used to run code simultaneously. Threads within a process share heap, text and data segments, etc with the exception that each thread has its own run-time stack.

It is recommended that you do not return a negative value from your thread's entry function, since a negative value distinguishes threads that are still running from those that have terminated.
