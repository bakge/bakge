Detailed Description

Windows in Bakge have their own contexts, but have a shared context across which they can share resources such as meshes, textures and more.

Result bakge::Window::Bind ( ) const
virtual

Only one OpenGL context can be current on any given thread. This method sets a window's context as current on the thread from which it is called. Any previous bound window's context will no longer be current on the thread.

Returns
BGE_SUCCESS if the context was successfully made current, and BGE_FAILURE if any error occurred.

Implements bakge::Bindable.

Result bakge::Window::Unbind ( ) const
virtual

This method removes the window's context as the current context on the thread from which it is called. The current context is set to none, and no code that requires an active OpenGL context can be safely run until a new Window is bound.

Returns
BGE_SUCCESS if the window context was removed as the current context, or BGE_FAILURE if any error occurred.

Implements bakge::Bindable.
