Detailed Description

Wrapper class for decoded Ogg Vorbis audio stream data. To play an audio Stream, you must first attach it to a Source and Play it.

See Also
Source, Source::Play

BGE_FACTORY Stream* bakge::Stream::Create ( int  Len,
Byte *  Data 
)

This factory function decodes a complete Ogg Vorbis stream and returns a Stream object that you can attach to a Source to play audio with. A Stream is not an audio stream in the traditional sense; the entire encoded audio file must be passed so it can be correctly decoded and turned into audio.

Returns
Pointer to allcoated Stream; NULL if any errors occurred or decoding failed.
